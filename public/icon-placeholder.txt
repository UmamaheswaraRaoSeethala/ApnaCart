ICON FILES REQUIRED FOR PWA INSTALLATION

To complete the PWA setup, you need to create the following icon files in the public/ directory:

1. favicon.ico (16x16, 32x32, 48x48)
2. icon-192x192.png (192x192 pixels)
3. icon-512x512.png (512x512 pixels)
4. apple-touch-icon.png (180x180 pixels)
5. apple-touch-startup-image-768x1004.png (768x1004 pixels)
6. apple-touch-startup-image-1536x2008.png (1536x2008 pixels)

RECOMMENDED ICON DESIGN:
- Use a simple, recognizable design
- Primary color: #22c55e (green)
- Secondary color: #3b82f6 (blue)
- White background for good contrast
- Simple vegetable or cart icon

ICON GENERATION TOOLS:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
- https://favicon.io/

For iOS splash screens, you can use:
- https://appsco.pe/developer/splash-screens

Place all generated files in the public/ directory.
